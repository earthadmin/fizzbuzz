# FizzBuzz Project
This is a simple javaScript program that prints numbers form 1 to 100 with the following rules: 
if a number is divisible by 3, it prints "fizz"
if a number is divisible by 5, it prints "buzz"
if a number is divisible by 15, it prints "fizzBuzz" 
otherwise  the number itself is pronted. 

## Example of the output is : 
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
